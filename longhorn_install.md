# Prerequisites

- install [open-iscsi](https://longhorn.io/docs/1.9.1/deploy/install/#installing-open-iscsi)
- [nfsv4 client](https://longhorn.io/docs/1.9.1/deploy/install/#installing-nfsv4-client)
- bash, curl, findmnt, grep, awk, blkid, lsblk
- [mount propagation](https://kubernetes-csi.github.io/docs/deploying.html#enabling-mount-propagation)
